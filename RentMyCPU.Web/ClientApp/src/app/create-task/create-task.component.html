To add a new task, please provide a valid .wasm file with one exported method int run(int param), which will be the entry point for your program. You can create a new wasm file with Webassembly Studio.

Run the task on {{ numberOfNodes }} nodes. An int parameter called param will be passed to your "run" method to indicate on which node your task is running.

Cost: 1 credit/hour/node.

If the computed task used more credits than you have, you will not be able to see the result of your task.

Tip: you can print char to the output of the program by including the following function inside your .c file (and calling it) : extern void writechar(char c);
